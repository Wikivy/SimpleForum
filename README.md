# SimpleForum

SimpleForum is an open-source forum extension for MediaWiki
providing Fandom-style discussion boards without proprietary
lock-in.

## Features
* Forum and thread namespaces
* Replies with signatures and timestamps
* Per-forum permissions
* Thread locking
* Sticky (pinned) threads
* Pagination and last-activity sorting
* REST API for forums, threads, replies, locks, and stickies
